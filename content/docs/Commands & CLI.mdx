# Commands & CLI
## Axiom++ Command Line Interface
```bash
axiom [command] [options]
```


--- 


## Commands
| **Command**     | **Description**            |
|-----------------|----------------------------|
| axiom init      | Initialize project         |
| axiom repl      | Launch interactive shell   |
| axiom verify    | Verify algebraic structure |
| axiom docs      | Generate documentation     |
| axiom benchmark | Run performance tests      |


## Example
```bash
axiom verify group config.ax
```

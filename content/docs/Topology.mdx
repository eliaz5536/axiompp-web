Topology is a branch of mathematics that studies the properties of spaces that are preserved under continuous transformations. It focuses on the concepts of continuity, compactness, connectedness, and convergence, among others.

Topology is essential in various areas of mathematics, including analysis, geometry, and algebra. It provides a framework for understanding the structure of spaces and the relationships between different mathematical objects.


# Compactness
Compactness is a topological property that generalizes the notion of closed and bounded subsets in Euclidean space. A topological space is compact if every open cover has a finite subcover. Compactness is an important concept in analysis and topology, as it allows for the extension of certain properties from finite sets to infinite sets.

```cpp
int main() {
    OpenSet<int> openSet({1, 2});
    std::cout << "OpenSet cardinality: " << openSet.cardinality() << "\n";
    std::cout << "OpenSet contains point 1: " << std::boolalpha << openSet.contains(1) << "\n";

    OpenCover<int> cover({openSet, OpenSet<int>({2, 3})});
    std::cout << "OpenCover covers space: " << std::boolalpha << cover.covers(space) << "\n";
    std::cout << "Compactness: " << std::boolalpha << Compactness<int>::isCompact(space, cover) << "\n";
    std::cout << "Connectedness: " << std::boolalpha << Connectedness<int>::isConnected(space) << "\n";
    
    return 0;
}
```

# Connectedness
Connectedness is a topological property that describes the extent to which a space is "in one piece." A topological space is connected if it cannot be divided into two disjoint non-empty open sets. # Continuity
Continuity is a fundamental concept in topology that describes the behavior of functions between topological spaces. A function is continuous if the preimage of every open set is open. Continuity is essential in understanding the relationships between different spaces and the behavior of functions defined on them.

```cpp

    ContinuousMap<int, int> identity(
        [](int x) { return x; },
        space,
        space
    );
    std::cout << "Continuity: " << std::boolalpha << identity.isContinuous() << "\n";

    return 0;
}
```

# Homeomorphism
Homeomorphism is a bijective continuous function between two topological spaces that has a continuous inverse. Homeomorphisms preserve the topological properties of spaces, making them equivalent in the sense of topology. Two spaces that are homeomorphic are considered to be "the same" from a topological perspective.


```cpp
int main() {
    ContinuousMap<int, int> forward([](int x) { return x; }, space, space);
    ContinuousMap<int, int> inverse([](int x) { return x; }, space, space);
    Homeomorphism<int, int> homeomorphism(forward, inverse);
    std::cout << "Homeomorphism: " << std::boolalpha << homeomorphism.isHomeomorphism() << "\n";    

    return 0;
}
```

# Homotopy
Homotopy is a concept in topology that describes a continuous deformation of one function into another. Two functions are homotopic if there exists a continuous family of functions that connects them. Homotopy is an important tool in algebraic topology, as it allows for the classification of spaces based on their topological properties.

```cpp
int main() {
    Homotopy<int, int> homotopy(
        forward,
        inverse,
        [](int x, double t) { return static_cast<int>(x + t); }
    );
    std::cout << "Homotopy: " << std::boolalpha << homotopy.isContinuous() << "\n";

    return 0;
}
```

# Path
Path is a continuous function from the unit interval [0, 1] to a topological space. Paths are used to study the connectedness and homotopy properties of spaces. A space is path-connected if any two points can be joined by a path.

```cpp
int main() {
    Path<int> path([](double t) { return static_cast<int>(t * 2.0); });
    std::cout << "Path start: " << path.start() << "\n";
    std::cout << "Path end: " << path.end() << "\n";
    std::cout << "Path loop: " << std::boolalpha << path.isLoop() << "\n";

    return 0;
}
```

# Path Connectedness
Path connectedness is a topological property that describes the ability to connect any two points in a space with a continuous path. A space is path-connected if for any two points, there exists a continuous function (path) that connects them. # Subspace Topology
